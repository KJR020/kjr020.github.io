# Source Definitions

Sources are grouped into categories. Each source has search queries and optional direct URLs for WebFetch.

## Category 1: AI Platform Official

Primary sources for AI model/API announcements and releases.

| Source | Search Query | Direct URL |
|--------|-------------|------------|
| Anthropic | `Anthropic Claude announcement {year}` | `https://www.anthropic.com/news` |
| OpenAI | `OpenAI announcement release {year}` | `https://openai.com/blog` |
| Google DeepMind / Gemini | `Google Gemini DeepMind announcement {year}` | `https://blog.google/technology/ai/` |

## Category 2: Developer Community & SNS

Trending discussions, tools, and projects from developer communities.

| Source | Search Query | Direct URL |
